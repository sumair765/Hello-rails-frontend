## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- NodeJS - [v16.x](https://nodejs.org/en/)
- NPM

### Setup

- git clone https://github.com/sumairq/Hello-rails-frontend.git
- cd hello-rails-frontend

### Install

- yarn

### Usage

- npm run start
